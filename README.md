# Installation
- add to your Cargo.toml:
```toml
...
[dependencies]
rsjson = "0.1.3"
```
- or run the following command in terminal:
```bash
cargo add rsjson
```

# Importation
- include the following line to your code
```rust
use rsjson;
```
